# GRIFFY ($GRFY)

Static GitHub Pages landing page for the GRIFFY Solana coin and NFT collection.

## Deploy on GitHub Pages

1. Push this folder to a public GitHub repository.
2. In the repository settings, enable GitHub Pages.
3. Set the Pages source to the branch and folder that contains `index.html`.

The page has no build step. It uses local image assets plus live DEX Screener Solana data for the `$GRFY` pair:

- Token mint: `D3SfZZMSkT1PUhkLvcYimVXTrSGwBw9vZcDH9VL822ch`
- Pair: `HMicZvhPJePexB8Szm1n57M3BskWNmiqgkqh6LZZ3Wpw`
